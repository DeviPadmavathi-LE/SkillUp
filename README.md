# SkillUp – Student Career & Learning Platform

SkillUp is a platform designed to help students discover job and internship opportunities, learn essential skills, and explore real interview experiences shared by selected candidates.

## Features
- Real-time job and internship alerts
- Skill-based filtering and opportunity suggestions
- Interview experience sharing and browsing
- Role-based learning paths
- Skill gap insights and guided learning recommendations
- Clean and user-friendly interface for better accessibility

##  Tech Stack
### Frontend

### Backend
